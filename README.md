## Short Nodes ใช้ class อะไรภูมิใจที่สุด เพราะอะไร

- class hidden lg:block เพราะ สามารถ ช่วยให้เว็บฉลาดขึ้น ในการแสดงผลบนหลายอุปกรณ์ เช่น แสดงText Logo เฉพาะ Dasktop และซ่อนบนipad
- class hover เพราะมันเป็นคลาสที่ช่วยให้ UI ของเว็บมีลูกเล่นมีชีวิตมากขึ้น และสร้างประสบการณ์ใช้งานที่ดีขึ้น
- Class grid เพราะสามารถช่วยจัด Layout ทั้งหมดให้เป็นระเบียบสวยงามและ Responsive
